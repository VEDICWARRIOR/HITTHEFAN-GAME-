# HIT THE FAN Roadmap

## Current Version

- Mobile-first 3D fan tapping game
- Randomized RPM limit per run
- Danger zone, screen shake, particles, smoke, debris, and destruction modal
- Persistent coins, best score, and best RPM
- Upgrade shop: Motor, Bearings, Luck
- Garage shop: fan skins and background themes
- Generated Web Audio sound effects
- GitHub-ready docs and publish checklist

## Next Improvements

- Add final custom fan models or optimized GLB assets
- Add achievement badges and daily reward streaks
- Add optional leaderboard service
- Add Android wrapper with Capacitor
- Add production app icons and store screenshots
- Add automated GitHub Pages deployment workflow
