# Contributing

Thanks for improving HIT THE FAN.

## Local Setup

```bash
npm install
npm run dev
```

## Before Opening a Pull Request

- Run `npm run build`.
- Test tapping, upgrades, garage purchases, retry, home, sound toggle, and vibration toggle.
- Keep changes focused and avoid committing generated `dist/` files unless a release workflow requires them.

## Code Style

- Use plain JavaScript modules.
- Keep UI changes mobile-first.
- Prefer Web Audio generated sounds unless a final licensed audio pack is added.
- Do not add paid or unlicensed assets.
